# Naidu Basudev — Portfolio

A heavy, light-themed React + Vite portfolio with Tailwind CSS and Framer Motion animations.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

Then open the URL printed in the terminal (default `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```

## Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Framer Motion
- React Icons
